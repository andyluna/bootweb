vsCode commit

1、只编译core模块
        递归安装core模块依赖的模块
        mvn clean install -pl com.andy.maven:core  -am
        递归安装cloud模块包含的子模块
        mvn clean install -pl com.andy.maven:cloud -amd


2、复制jar文件    mvn dependency:copy-dependencies
   查看jar文件    mvn dependency:tree
   复制jar到目录  mvn dependency:copy-dependencies -DoutputDirectory=target/lib

3、不递归子模块
    mvn clean install -N

4、maven安装jar包 到本地仓库
     mvn install:install-file -Dfile=D:\database\oracle11.2.0\product\11.2.0\dbhome_1\jdbc\lib\ojdbc6.jar -DgroupId=com.andy.jdbc -DartifactId=oracle -Dversion=1.0.0 -Dpackaging=jar

5. package -D maven.javadoc.skip=true

6.注解事物 如果采用JPA原生的 注解
   编译时会报错
   需要在maven里面再导入
   <dependency>
       <groupId>javax.interceptor</groupId>
       <artifactId>javax.interceptor-api</artifactId>
       <version>1.2</version>
   </dependency>
   <dependency>
       <groupId>javax.enterprise</groupId>
       <artifactId>cdi-api</artifactId>
       <version>1.2</version>
       <scope>provided</scope>
   </dependency>

7. maven doc 插件输出乱码
-Dfile.encoding=GBK

8.mvn 切换 profile配置
    mvn clean compile -Pdev
